Hospital Nurse

Vitals capture and the nurse station, with automatic triage escalation

Hospital Nurse is the nurse-station addon of the Hospital Management System suite, built on ``hospital_base`` and ``hospital_reception``. It gives nursing staff a fast, tablet-first way to capture vitals and automatically routes abnormal readings to the doctor with the correct priority -- no manual escalation step required.

Key Capabilities

  • Structured vitals capture with auto-computed, color-flagged BMI.
  • Clinically-informed range validation that rejects obvious data-entry errors.
  • Automatic priority escalation to Urgent on abnormal vitals -- never de-escalates an existing Emergency.
  • Automatic visit transition from "Waiting for Nurse" to "Waiting for Doctor" the moment vitals are saved.
  • A tablet-first Nurse Dashboard: next patient card, one-tap vitals entry, and the nurse task checklist.
  • A simple OPD nurse task checklist, ready to be extended by the future IPD module for ward rounds.

Built for the Full Suite

Hospital Nurse extends ``hospital_base``'s visit model via standard Odoo inheritance and reuses its audit mixin and security groups -- keeping the whole hospital_* suite consistent and upgrade-safe.
