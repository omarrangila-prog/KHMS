Hospital Executive Dashboard

Cross-department, admin-level KPI rollup for the Hospital Management System. Per-department dashboards (reception, nurse, doctor, pharmacy, lab, radiology, ward) each ship in their own module; this module adds the executive cross-cutting view on top of all of them.

Key Features

  • Patients seen today / this week.
  • Bed occupancy percentage across all wards.
  • Average doctor wait time (check-in to first consultation).
  • Ward revenue from confirmed discharges (length-of-stay billing).
  • All KPIs computed in PostgreSQL via a SQL-view-backed model, not looped in Python.

Dependencies

hospital_base hospital_reception hospital_doctor hospital_pharmacy hospital_lab hospital_radiology hospital_ipd
