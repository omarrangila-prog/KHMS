Hospital Radiology

Full radiology order and result management for the Hospital Management System. Mirrors the hospital_lab module structure for imaging modalities: X-Ray, CT, MRI, Ultrasound, and Mammography.

Key Features

  • Radiology Study Catalog with modality classification.
  • Order state machine: Ordered → Scheduled → In Progress → Completed.
  • Priority-driven queue (Normal / Urgent / Emergency) with colour-coded badges.
  • Findings Entry: narrative findings + impression text + multi-file attachment upload.
  • Automatic doctor notification (mail activity) on order completion.
  • Cancel wizard requiring a documented reason (audit trail).
  • Visit aggregate-state integration: visit moves to Billing only after all radiology orders complete.
  • Multi-company isolation via company record rules.
  • QWeb Radiology Report PDF with findings, impression, and attachments list.

Dependencies

hospital_doctor (brings in hospital_nurse, hospital_reception, hospital_base).

Default Study Catalog

Includes 5 standard imaging studies installed in every environment:

  • Chest X-Ray (PA View)
  • CT Head (Non-Contrast)
  • MRI Spine (Lumbar)
  • Ultrasound Abdomen
  • Mammography (Bilateral)
