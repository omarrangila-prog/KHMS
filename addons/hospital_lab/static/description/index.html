Hospital Laboratory

Full laboratory order and result management for the Hospital Management System. Covers the complete order-to-result lifecycle from doctor ordering through sample collection, analysis, and result delivery.

Key Features

  • Lab Test Catalog with normal reference ranges (per parameter).
  • Lab Order state machine: Ordered → Sample Collected → Processing → Completed.
  • Priority-driven queue (Normal / Urgent / Emergency) with colour-coded badges.
  • Structured Result Entry: multi-parameter grid with is_abnormal auto-flag.
  • Automatic doctor notification (mail activity) on order completion.
  • Cancel wizard requiring a documented reason (audit trail).
  • Visit aggregate-state integration: visit moves to Billing only after all lab orders complete.
  • Multi-company isolation via company record rules.
  • QWeb Lab Report PDF with abnormal result highlighting.

Dependencies

hospital_doctor (brings in hospital_nurse, hospital_reception, hospital_base).

Default Catalog

Includes 6 standard lab tests installed in every environment:

  • CBC (Complete Blood Count)
  • Blood Glucose Fasting (FBS)
  • Lipid Panel
  • Urine Routine Examination
  • Liver Function Tests (LFT)
  • Kidney Function Tests (KFT)
