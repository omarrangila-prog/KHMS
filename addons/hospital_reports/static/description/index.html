Hospital Reports

Centralised report branding configuration and cross-module aggregate printable reports for the Hospital Management System.

Key Features

  • Per-company report branding: logo, header/footer text, accent colour.
  • Shared QWeb layout inherited by all hospital report templates.
  • Patient Visit History report (PDF, printable from patient form).
  • Daily Census report (date-range wizard, PDF).
  • Ward Occupancy report (real-time bed state snapshot, PDF).
  • Settings → Report Branding configuration menu (admin-only).

Dependencies
